# Classic-crypto

## Tietorakenteet ja algoritmit aineopintojen harjoitustyö

Harjoitustyö toteuttaa Java-kielellä klassisia (pre-1970-luku) kryptografisia algoritmeja ja kryptoanalyysiä näihin liittyen.
